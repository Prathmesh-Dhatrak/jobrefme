Installation Guide

Installing the Chrome Extension

1

Download the Extension

Download the latest release of JobRefMe extension from GitHub.

Download JobRefMe Extension
2

Extract the ZIP File

Extract the downloaded ZIP file to a location on your computer that you won't accidentally delete.

3

Load in Chrome

Open Chrome and navigate to the extensions page by typing chrome://extensions/ in the address bar.

  • Enable "Developer mode" in the top-right corner
  • Click "Load unpacked" and select the extracted folder
  • The JobRefMe extension should now appear in your extensions list
4

Get a Gemini API Key

You'll need a Google Gemini API key to use this extension.

  1. Visit Google AI Studio
  2. Sign in with your Google account
  3. Go to "API keys" section
  4. Click "Create API Key"
  5. Copy the generated key for use in the extension
5

Complete Initial Setup

After installation, a welcome page will open automatically. Enter your Google Gemini API key and click "Save API Key".

How to Use JobRefMe

Method 1: HireJobs.in Integration

  1. 1

    Navigate to a job posting on HireJobs.in (e.g., https://hirejobs.in/jobs/abc123)

  2. 2

    Click on the JobRefMe extension icon in your browser toolbar

  3. 3

    Click the "Generate Referral Request" button

  4. 4

    Wait while the AI generates your personalized message

  5. 5

    Click "Copy" to copy the message to your clipboard

Method 2: Universal Content Selection

  1. 1

    On any job posting website, select the job description text

  2. 2

    Right-click on the selected text

  3. 3

    Choose "Generate Referral from Selected Content" from the context menu

  4. 4

    In the popup, click "Generate Referral Request"

  5. 5

    Copy the generated message to use in your referral request

Pro Tips

  • Sign in with your Google account to save your templates and settings
  • Create custom templates for different types of roles or companies
  • You can regenerate a referral message if you're not satisfied with the initial result
  • The extension works best with detailed job descriptions that include skills and qualifications

Frequently Asked Questions

Do I need to pay for using JobRefMe?

JobRefMe is completely free to use. However, you'll need your own Google Gemini API key, which has its own pricing model. Google typically offers free credits for new users.

Which job boards are supported?

JobRefMe currently has direct integration with HireJobs.in. However, you can use the universal content selection method on any job board by selecting the job description text.

Is my data secure?

Yes, your privacy is important to us. Your Gemini API key is stored securely in your browser's local storage, and no personal data is collected or sent to our servers.

Can I customize the generated messages?

Yes, you can edit the generated messages before copying them. Additionally, if you sign in with your Google account, you can create and manage custom templates.

How do I report bugs or suggest features?

You can report bugs or suggest features by opening an issue on our GitHub repository.
